## 泄露内容详解

### 核心代码模块

| 模块 | 行数 | 功能 |
|------|------|------|
| API 调用引擎 | 46,000 | AI 对话核心 |
| 工具定义系统 | 29,000 | 40+ 权限门控工具 |
| IDE 桥接 | - | VS Code / JetBrains 完整逻辑 |
| Agent 编排 | - | 多 Agent 并行任务系统 |

### 技术亮点

1. **System Prompt 工程**
   - 分段构建
   - DYNAMIC_BOUNDARY 缓存边界
   - 行为引导技巧

2. **上下文管理**
   - Token 预算三函数
   - 四级告警机制
   - Microcompact / Full Compact / Session Memory Compact

3. **Prompt Cache**
   - cache_control 放置策略
   - CacheSafeParams
   - byte-exact fork threading

4. **Thinking 控制**
   - ThinkingConfig 三模式
   - Effort 四级
   - ultrathink 关键词触发
   - Advisor 机制

5. **工具系统**
   - buildTool() 抽象
   - Tool 接口 30+ 方法
   - builder 模式
   - 三层条件注册漏斗
   - ToolSearch 延迟加载

6. **BashTool**
   - 12400 行代码
   - 四层安全纵深
   - 沙箱执行
   - sed 编辑特殊处理

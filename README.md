## 🎯 核心方法论

基于 [Karpathy LLM Wiki](https://gist.github.com/karpathy/442a6bf555914893e9891c11519de94f) 理念：

1. **raw/** — 原始资料（不可变）
2. **wiki/** — LLM增量编译的交叉链接文档
3. **CLAUDE.md** — 入口Schema

### 第2章：水凝胶冻结的基础科学

**写作时机：** 第1轮（建立全书理论根基）
**预计字数：** 英文 3000-4000 words

**小节结构：**
```
2.1 水在水凝胶网络中的状态
    - 自由水 vs 结合水 vs 中间水（DSC/TGA区分）
    - 纤维素-OH强H-bond → 不可冻结合水比例显著高于合成聚合物
    - 定量数据：典型纤维素水凝胶的不可冻结水含量范围

2.2 冰晶形成动力学
    - 均相成核 vs 异相成核
    - 纳米约束空间内的Gibbs-Thomson效应
    - 纤维素纳米纤维孔道对冰晶生长的几何限制

2.3 冻结诱导的级联失效（统一损伤模型，本文原创概念）
    冰核形成 → ①孔结构破坏 → 冰塌陷
            → ②离子通道中断 → 电导骤降
            → ③链应力集中 → 脆性断裂
            → ④界面水冻结 → 粘附丧失
    ︿—需做原创示意图1：四路级联损伤—﹀

2.4 评价宽温域性能的标准框架
    - 现有方法的问题：只报"最低可耐受温度"，不报该温度下的定量功能数据
    - 证据：174篇笔记中仅9篇同时报告低温+导电数据
    - 建议框架：Tier 1（-20°C功能完整）→ Tier 2（-40°C）→ Tier 3（-60°C）→ Tier 4（-80°C）
    ︿—需做原创表1：现有文献的低温度量功能数据覆盖率统计—﹀
```

### 第3章：盐类抗冻 — 依数性冻结降低

**写作时机：** 第2轮
**预计字数：** 英文 4000-5000 words

**小节结构：**
```
3.1 物理化学基础
    - van't Hoff: ΔT_f = K_f·m·i
    - 盐在水凝胶中的有效浓度（≠ 配制浓度，因为纤维素H-bond竞争）
    - Hofmeister序列与纤维素网络的相容性

3.2 主流盐体系系统对比
    表：LiCl/CaCl₂/ZnCl₂/NaCl/KCl/KAc → T_f，σ，纤维素相容性，成本
    ← 从24篇提取数据填空 →

3.3 盐-纤维素协同机制
    - LiCl：Li⁺水化半径大 → 强结构破断 → 高ΔT_f
    - CaCl₂：Ca²⁺双电荷 → 离子交联增强网络
    - ZnCl₂：特殊，同时溶纤 + 抗冻
    - 案例精析：每类盐选1-2篇标杆

3.4 盐的三重角色悖论（本综述设计原则#2）
    抗冻 ← 高浓度 → 好
    导电 ← 中浓度 → 好
    网络不破坏 ← 低浓度 → 好
    → "甜点浓度"概念：三类性能需求的交集区间
    ︿—需做图：三重角色悖论的示意图—﹀

3.5 多盐混合策略
    - LiCl + CaCl₂ → 协同抗冻
    - 盐 + 有机溶剂 → 突破单一盐的极限
```

**需要的数据表：**
| 表 | 内容 | 状态 |
|----|------|------|
| 表3.1 | 盐体系统一对比例（5种盐 × 6个性能维度） | 需从24篇提取 |
| 表3.2 | LiCl浓度-ΔT_f-σ关系（含甜点标注） | 需提取 |
| 表3.3 | 代表性盐基水凝胶性能汇总 | 从笔记提取 |

### 第4章：DES/离子液体抗冻 — 氢键竞争与冰核干扰

**写作时机：** 第3轮
**预计字数：** 英文 5000-6000 words
**重要度：** ★★★★★（综述的核心差异化章）

**小节结构：**
```
4.1 DES/IL的物理化学本质
    - 非依数性机制：H-bond竞争 → 冰核形成受阻
    - 与盐类的根本区别：
        盐：溶质与水竞争（不改变溶剂主体结构）
        DES：整体溶剂重构（改变了水的热力学环境）
    - 相图分析：DES-水的伪二元相图

4.2 三类主流DES/IL体系
    ZnCl₂基：T_f≤-80°C，溶纤最优，酸性限制
    ChCl基：T_f≤-80°C，绿色无毒，高黏度
    Betaine基：T_f≤-60°C，生物相容，成本高
    
    每类：HBA/HBD结构 → 黏度 → 电导 → 抗冻的构效关系
    ← 核心对比表：三类DES × 8个性能维度 →

4.3 DES对纤维素的独特化学（原创核心观点）
    "一剂三效"假说：
    ① 溶纤 — HBD（如H₃PO₄、urea）中的-OH/NH₂与纤维素-OH竞争H-bond
    ② 抗冻 — HBA（如Zn²⁺、Ch⁺）与水形成稳定复合物
    ③ 导电 — DES自身离子载体不需要额外盐
    ← 这是本文能从已有综述中脱颖而出的核心argument →

4.4 标杆案例深度剖析
    ZnCl₂/H₂O/H₃PO₄ DES + cellulose：
    - 4086% 拉伸 → 韧性261.3 MJ/m³
    - 1.64 S/m @20°C, 0.9-1.37 @15°C
    - GF 5.4, ≤-80°C, >85%透明
    - 同时实现自修复
    
    为什么这个体系如此成功？
    → 分析：DES同时满足三个维度的需求，没有出现"此消彼长"

4.5 设计原则：DES选择矩阵
    三维决策空间：
    X轴：温度目标（-20°C ~ -80°C）
    Y轴：纤维素溶解需求（不溶/溶胀/溶解）
    Z轴：功能需求（导电/透明/自修复）
    ← 每个文献(doi, 策略, 性能)被映射到这个3D空间 →
    ︿—需做图：DES选择矩阵3D示意图—﹀

4.6 未来方向
    - 功能解耦（当前最大瓶颈：三个功能耦合度过高）
    - 计算化学预测DES组成
```

**需要的数据表：**

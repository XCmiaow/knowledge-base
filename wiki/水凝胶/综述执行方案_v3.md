## 五、语言降噪清单

**替换规则（全篇搜索替换）：**

| 删除/替换 | 原因 |
|----------|------|
| "promising class" → 删除或用具体数据支撑 | 空泛 |
| "unparalleled platform" → "chemically versatile platform" | 过度断言 |
| "catastrophic loss" → "severe degradation" | AI味 |
| "burgeoning applications" → "emerging applications" | 模板语言 |
| "sophisticated strategies" → 删掉，直接讲策略 | AI味 |
| "remarkable synergy" → "coupled improvement" | 过度形容词 |
| "paradigm shift" → 删除或替换为"shift in design logic" | 最危险的空话 |
| "poised to play a pivotal role" → "may enable specific applications if..." | 加上边界条件 |
| "these materials are set to revolutionize" → 删除 | overclaim |
| "unprecedented" → 除非有历史数据证明，否则删除 | 危险词 |

**关键科学准确性修正：**
- 每个"biocompatible" → 引用具体测试数据，或加注"requires further evaluation"
- 每个"green/sustainable" → 区分"cellulose-based component" vs "entire gel system"
- 每个"ultra-low temperature" → 注明具体温度和功能状态

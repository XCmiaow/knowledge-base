# 纤维素基宽温域水凝胶综述 — 执行方案 v4（基于 Pilot 真实数据）

> **核心转变：** v3 的"数据驱动综述"假设错了——摘要级数据支持不了定量对比。v4 将这一发现本身转化为综述的核心贡献。

## 零、Pilot 实验结论（n=20，随机抽样）

### 可提取率
| 字段 | 可提取率 | 评价 |
|------|---------|------|
| 策略分类 | 70% | 可做定性分析 |
| 纤维素类型 | 85% | 可做角色矩阵 |
| 拉伸率 | 50% | 碎片化，不可比 |
| 拉伸强度 | 40% | 碎片化 |
| 室温电导 | 25% | 低，无可比性 |
| **GF** | **0%** | **完全不可行** |
| **T_claimed** | **5%** | **接近不可行** |
| **T_functional** | **0%** | **完全不可行** |
| 含水量 | 0% | 完全不可行 |
| 证据等级 D | 19/20 (95%) | 领域的数据报告标准缺失 |

### 根本结论

**摘要级文献记录在定量功能数据维度上几乎是空的。**
不是"有些没报"，而是"几乎都没报"。

### 这对综述意味着什么

```
放弃的 → 从174篇提取定量数据、画Pareto气泡图、NR热力图、跨策略性能对比表
保留的 → 定性策略分类、机制对比、纤维素角色矩阵、应用分级
转为核心贡献 → 将"数据缺失"从一个小发现升级为全文核心论点
```

## 一、修订后定位

### 核心叙事（三个金句）

**1. The claimed temperature window is not the functional temperature window.**
声称的耐受温度窗口 ≠ 真实功能温度窗口。

**2. Cellulose is an active regulator of water state, network mechanics, and interfacial interactions — not merely a sustainable filler.**
纤维素不是绿色填料。

**3. The field needs standardized reporting of functional retention at claimed temperatures — without this, cross-study comparison is impossible.**
领域最大瓶颈不是缺材料，而是缺可比证据。

### 目标期刊：Advanced Functional Materials（综述栏目）

**理由不变**：功能性能视角与AFM偏好对齐。但篇幅缩减至30-40页（数据驱动的定量部分取消，改为机制驱动的批判性分析）。

### 写作角度

从"数据驱动综述"调整为 **"机制驱动 + 证据批判 + 标准化框架建议"**

核心贡献从"我比较了所有策略" 变成 **"我先证明目前无法公平比较，然后提出怎么解决这个问题"**。

## 九、v4 的自我批评（在写之前就承认）

1. **缺乏定量对比是硬伤。** 不管怎么包装，"不能做定量Pareto分析"就是比能做定量分析的综述弱。这是数据可得性瓶颈，不是框架设计问题。

2. **"证据缺失作为核心发现"的方法论风险。** 审稿人可能问："你没查到≠不存在"。缓解：①声明搜索范围+方法 ②在全文note中约定论调为"at the abstract level, the reporting is incomplete" ③对全文文献也要求同样诚实。

3. **cellulose-specific contribution 需要非常小心论证。** 第三章如果写成"纤维素很好，所以什么都好"，会被审稿人一票否决。必须区分"纤维素实际贡献了什么"vs"换成PVA也能做出的东西"。

4. **140篇无摘要文献仍是一个大漏洞。** 如果这些里面恰好有很多低温功能数据，全部分析的基础就动摇了。需要明确：我们分析的是abstract-level reporting completeness，不是全文级。

## 一、定位修正

### 文章类型
**Critical scoping review**（不是 systematic review，也不是 quantitative meta-review）

### 核心叙事（三个金句，v5版）

**1. The claimed temperature window is rarely the verified functional temperature window at the abstract level.**
在摘要层面，声称的温度窗口很少等于验证的功能温度窗口。

**2. Cellulose is an active regulator of water state, network mechanics, and interfacial interactions — not merely a sustainable additive.**
纤维素是主动调控因子。

**3. The field's current reporting conventions hinder cross-study comparison. The most urgent need is not lower temperatures, but a standardized framework for reporting functional retention under thermal stress.**
领域最紧迫的任务不是继续下探更低的温度，而是建立标准化的功能保持报告框架。

### 目标期刊
**Advanced Functional Materials**（综述栏目），或 **ACS Nano Perspective**（15页精简版作为备选）

## 二、双层证据体系（方法论核心）

### Reading depth

| Level | 定义 | 本文数量 | 可用于 |
|-------|------|---------|--------|
| **R1** | Full-text extracted via Codex | 3 篇 | 案例分析、定量证据、机制细节 |
| **R2** | Abstract-level screened via MiMo | 111 篇 | 策略分布、趋势统计、摘要层面报告完整性 |
| **R3** | Metadata only (DOI/title/journal) | 140 篇 | 领域总量背景、期刊分布 |

### Functional evidence level（仅用于 R1 文献）

| Level | 定义 |
|-------|------|
| **F4** | Low-T function + cycling/duration + device validation |
| **F3** | Low-T functional measurement (conductivity, GF, etc.) |
| **F2** | Room-T function + low-T structural survival (no function test at low T) |
| **F1** | Structural anti-freezing only (DSC, bending photo) |
| **F0** | No functional evidence visible |

### 证据使用边界表（方法部分必须包含）

| 信息类型 | R1可用 | R2可用 | R3可用 | 可否用于定量 |
|---------|--------|--------|--------|------------|
| 年份/期刊/材料类别 | ✅ | ✅ | ✅ | 趋势，不可排名 |
| 策略分类（盐/DES/有机/网络） | ✅ | ✅ | ❌ | 粗分类 |
| 具体性能数值（电导/强度/GF） | ✅ | ⚠不可靠 | ❌ | **仅R1** |
| T_functional | ✅ | ⚠不可判断 | ❌ | **仅R1** |
| 机制判断 | ✅ | ⚠谨慎 | ❌ | **主要R1** |
| 应用成熟度 | ✅ | 可初判 | ❌ | 深度需R1 |
| 纤维素角色分类 | ✅ | ✅ | ❌ | 定性分类 |

## 七、v5 的自我批评

**1. R1 = 3 篇是致命短板。**
无法声称"代表性全文分析"。只能诚实写："We illustrate key design principles through detailed analysis of selected full-text cases, while using abstract-level screening to map the broader landscape."

**2. "263篇R1/R2/R3" 不等于 "我读了263篇"。**
140篇R3我连摘要都没见过。111篇R2我只读了摘要。3篇R1我也只是让Codex跑了一遍。必须在方法部分严格区分。

**3. 这个综述的核心价值不在"全面"，而在"诚实的分层方法和评价框架"。**
如果审稿人想要一份"穷尽了所有文献所有数据的定量综述"，v5做不到。但如果审稿人接受一个"指出领域当前证据标准不足、并提出改进框架"的critical review，v5能做。

**4. 最坏情况：**
如果审稿人说"你只有3篇全文，凭什么写综述"——这个风险真实存在。唯一防御是：文章定位是critical scoping review + framework proposal，不是数据驱动的comprehensive review。这必须在introduction和methodology里非常明确。
